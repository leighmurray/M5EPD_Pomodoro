# M5EPD M5Paper Pomodoro

This is a simple program that lets you use your M5Paper as a Pomodoro timer. Default time is 15 minutes.

Start -> start timer.

Stop -> stop timer.

Reset -> reset timer.

Reset when timer is already reset -> reset tomato counter.
